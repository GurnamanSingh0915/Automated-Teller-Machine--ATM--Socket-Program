Steps to run the Automated Teller Machine (ATM) Socket Program =>

1) Copy files from local system to FileZilla, depending on whether the files will be executed on local system or Putty server.
2) Open a command prompt or terminal on your system or putty. 
3) Type in python3 server.py to start the server on one terminal.
4) Type in python3 client.py to start the client on another terminal. Make sure to close existing client processes to avoid mixups.  
5) Enter the option displayed by the client to process an action (Withdraw, Deposit, or Check Balance).
6) Values involving characters or float values are not allowed.
7) Typing in '0' to exit will stop the server and client process.
